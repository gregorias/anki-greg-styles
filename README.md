# Anki Greg Styles

An Anki plugin that manages my card styles.

## For Developers

### Dev Environment Setup

Set up Pipenv:

    pipenv install --dev

Set up npm:

    npm install

Install Lefthook:

    lefthook install

## Release & Installation

### Build & import

1. Run `package`.
2. Import `ankiassets.ankiaddon` in Anki.
